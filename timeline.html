  11. Code for Good by J.P.Morgan

    November 12th, 2016

    Code for good 2016 was organised in Glasgow by J.P.Morgan. My team and I coded over the weekend for 24 hours an application for a charity aiming to teach children safety on the road. We used Java and Reactivision, a vision recognision library with special markers to create an application that launched certain animations when specific markers were shown to the camera. The project is to be further developed and could be displayed on the floor with the help of a projector. The children could play interactively as lolypop men, drivers, pedestrians and bike riders to learn rules and safety on the road

  12. Hashcode by Google

    February 16th, 2017

    My team and I created a Java application that optimised cache memory for youtube depending on the number of requests from the endpoints. The program scans a file with data on the number of videos, sizes, cache spaces and latencies and assigns videos to caches in the most efficient manner

  13. CodeTheCity

    February 28th, 2017

    My team and I created NoBot, a chatbot that prevents unnecessary meetings saving the company money and time in a sarcastic manner. It logs minutes and agendas and sends them to the team members at the end of the meeting. Further development would involve connecting it to Google Calendar to check availability and book meetings. Done using Slack in node.js with a redis database for agendas
